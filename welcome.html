Blogger Api
User route
  • Get => /users
  • Get user => /user/:id
  • Patch => /user/edit
  • Patch => /user/changepassword
Post route
  • Get => /posts
  • Get post => /posts/:id
  • Post => /posts
  • Patch => /posts/:id
  • Delete => /posts/:id
Auth route
  • Post => /login
    for subsequent request after /login append the token in the headers e.g headers: { Authorization: 'Bearer' + token }
  • Post => /signup
